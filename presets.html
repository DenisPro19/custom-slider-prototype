Game Presets

CMS › Game Presets demo: reset
Prototype. A preset is a saved game-filter configuration (scope, provider/type/category, include/exclude, pins, order, sort), not just a flat list. You build it with the same picker used in the Game Filter, save it once, and reuse it in categories, achievements and other modules. Editing a preset updates every place that links it. Presets are per product (casino / sweepstakes have separate catalogs). Look and details will change: this is a sketch ahead of the final design.
