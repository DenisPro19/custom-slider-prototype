CMS › Game Presets demo: reset
Prototype. A preset is a game list you build once and reuse anywhere a game list is needed: lobby categories (Game Filter) and achievements. Editing a preset updates every place that links it; if you only want the change in one place, save your edits as a new preset. The interface will still change.
